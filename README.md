# Stay Connected

A simple tool to prevent Windows from automatically disconnecting your Wi-Fi when it thinks the internet is slow or blocked.

## Problem

Windows sometimes checks internet connectivity by connecting to Microsoft servers. When these servers are slow or inaccessible (especially with international connection issues), Windows incorrectly assumes the network has no internet and may disconnect your Wi-Fi.

## Solution

This tool modifies Windows network connectivity check settings to stop this automatic behavior.

## Download

Go to [Releases](https://github.com/bardianz/stay-connected-windows/releases) and download the latest `.exe` file.

## How to Use

1. Right-click on the downloaded file
2. Select **"Run as administrator"**
3. Choose option 1 to fix the problem
4. Restart your computer

## Options

- **Option 1:** Fix the problem - Prevents Windows from checking internet status
- **Option 2:** Restore defaults - Reverts to original Windows settings
- **Option 3:** Exit

## Requirements

- Windows 10 or Windows 11
- Administrator privileges
