# اStay Connected

ابزاری ساده برای جلوگیری از قطع خودکار وای‌فای توسط ویندوز، زمانی که تصور می‌کند اینترنت کند یا مسدود شده است.

## مشکل

ویندوز گاهی اوقات برای بررسی اتصال اینترنت، به سرورهای مایکروسافت متصل می‌شود. وقتی این سرورها کند یا در دسترس نباشند (مخصوصاً در مشکلات اینترنت بین‌الملل)، ویندوز اشتباهاً تصور می‌کند شبکه اینترنت ندارد و ممکن است وای‌فای را قطع کند.

## راه‌حل

این ابزار تنظیمات بررسی اتصال اینترنت ویندوز را تغییر می‌دهد تا از این رفتار خودکار جلوگیری کند.

## دانلود

به بخش [Releases](https://github.com/bardianz/stay-connected-windows/releases) بروید و آخرین فایل `.exe` را دانلود کنید.

## طریقه استفاده

۱. روی فایل دانلود شده راست‌کلیک کنید
۲. گزینه **"Run as administrator"** (اجرا به عنوان مدیر) را انتخاب کنید
۳. گزینه ۱ را برای رفع مشکل انتخاب کنید
۴. کامپیوتر خود را مجدداً راه‌اندازی کنید

## گزینه‌ها

- **گزینه ۱:** رفع مشکل - جلوگیری از بررسی وضعیت اینترنت توسط ویندوز
- **گزینه ۲:** بازگرداندن به حالت اولیه - برگرداندن تنظیمات اصلی ویندوز
- **گزینه ۳:** خروج

## نیازمندی‌ها

- ویندوز ۱۰ یا ویندوز ۱۱
- دسترسی مدیر (ادمین) سیستم
